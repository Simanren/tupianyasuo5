# 图片压缩工具

一个简洁优雅的在线图片压缩工具,采用苹果设计风格。

## 功能特点

- 支持上传 PNG、JPG 等格式图片
- 支持自定义压缩比例
- 实时预览压缩前后的图片效果
- 显示压缩前后的文件大小
- 支持压缩后图片下载
- 优雅的苹果风格界面设计

## 技术栈

- HTML5
- CSS3 
- JavaScript
- 使用浏览器原生的图片压缩API

## 页面结构

```
index.html          // 主页面
assets/
  ├── css/          // 样式文件
  │   └── style.css
  └── js/           // JavaScript文件
      └── main.js
```

## 使用说明

1. 点击上传区域或拖拽图片到页面
2. 选择需要的压缩比例
3. 预览压缩效果
4. 点击下载按钮保存压缩后的图片
